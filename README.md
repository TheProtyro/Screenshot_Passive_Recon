# Screenshot_Passive_Recon
A tool used to screenshot the passive reconnaissance part of an external or web pentest.

## Dependencies

```
pip3 install -r requirements.txt
```

## Usage

```
python3 screenshot_passive_recon.py -d domain.tld
```
